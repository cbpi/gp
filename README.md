# 股票历史查询

## 安装依赖
```
npm install
```

### 运行项目
```
npm run serve
```

### 打包
```
npm run build
```

### Mac OSX启动MongoDB服务
```
npm run mongodb
```

### 启动MongoDB
```
node server/index.js
```


技术栈：Vue3+MongoDB
